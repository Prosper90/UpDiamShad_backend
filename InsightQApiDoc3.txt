Create a connection link
post
https://api.staging.insightiq.ai/v1/links
Creates the microsite link to initiate account linking.

Request
Basic authentication is a simple authentication scheme built into the HTTP protocol. To use it, send your HTTP requests with an Authorization header that contains the word Basic followed by a space and a base64-encoded string username:password.

Example: Authorization: Basic ZGVtbzpwQDU1dzByZA==

Body

application/json

application/json
name
string
Name of the creator

external_id
string
required
A unique ID sent by the developer to identify a creator (this should be unique in the developer's system)

Responses
201
Returns the microsite URL along with an invite ID

Headers
request-id
string<uuid>
required
Unique ID associated with this request. Please quote this request while requesting support.

Body

application/json

application/json
responses
/
201
created_at
string<date-time>
required
Object creation timestamp (UTC date and time in ISO 8601 format)

Example:
2020-03-27T12:56:34.534978
microsite_url
string<uri>
required
Unique link which has the Connect SDK embedded in it for an optimized account connection flow.

invite_id
string<uuid>
required
Unique invite ID generated by InsightIQ

external_id
string
required
The unique ID sent by the developer to identify the creator

9  Retrieve an account
get
https://api.staging.insightiq.ai/v1/accounts/{id}
Retrieve the information of the Account with the supplied Account id. An Account represents the user account on the work platform end. A user can connect multiple account in the same or across multiple work platforms.

Usually used in conjunction with the "ACCOUNTS.*" webhooks, read more.

Request
Basic authentication is a simple authentication scheme built into the HTTP protocol. To use it, send your HTTP requests with an Authorization header that contains the word Basic followed by a space and a base64-encoded string username:password.

Example: Authorization: Basic ZGVtbzpwQDU1dzByZA==

id
string
required
Unique identifier for the account.

Responses
200
Returns Account object.
